# AD00007-REVIVE-USB

## オリジナルUSBデバイスを簡単に作成できるREVIVE USB  
![](http://bit-trade-one.co.jp/wp/wp-content/uploads/2014/03/d3bd346bc2119d8576ce93741d53400c.png)  

USBでパソコンと接続し、専用ソフトウェアを用いて12本のピンに  
任意の操作（マウスクリック／キー入力等）を設定する事が出来ます。
